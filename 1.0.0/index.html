Gesture Flick

Figure out event type and direction. by event data.

Feature

  • 터치이벤트 좌표들을 통해 이동방향을 추적한다.
    • N/E/S/W
    • NE/ES/SW/WN
  • 넘겨받은 이벤트데이터로 이벤트의 종류를 추적한다.
    • click
    • dbclick
    • flick
