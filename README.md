Gesture Reader
===============
제스처 리더 컴포넌트<br>이벤트 또는 포인트 좌표로 이벤트의 타입과 종류를 판별한다.

## Feature
* Touch 또는 Click 이벤트의 좌표로 사용자 제스쳐의 방향을 판독
	* N/E/S/W
	* NE/ES/SW/WN
* 이벤트 데이터로 사용자 제스쳐의 종류를 판독
	* click/tab
	* dbclick/dbtab
	* flick
